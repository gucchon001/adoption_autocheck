# AIチャットアプリケーション機能要件仕様書

## 1. システム概要
このアプリケーションは、OpenAIのAPIを利用して、Pythonコードのリファクタリング提案や仕様書の生成を行うためのツールです。Pythonファイルを統合し、リファクタリングの改善点をAIから取得し、それを基に詳細な仕様書を生成します。システムはコマンドラインから実行され、ユーザーは実行したい機能を選択できます。

## 2. 主要機能要件
1. **Pythonファイルのマージ**:
   - 指定されたディレクトリ内のPythonファイルを統合し、特定のパターンに基づく除外を行います。
   - 統合結果は`merge.txt`として保存されます。

2. **仕様書の生成**:
   - 統合されたPythonコードを解析し、AIを利用して機能要件仕様書を生成します。
   - 生成された仕様書は`requirements_spec.txt`として保存されます。

3. **詳細なプログラム仕様書の生成**:
   - 機能要件仕様書と統合されたコードを基に、AIを利用して詳細なプログラム仕様書を生成します。
   - 生成された仕様書は`detailed_program_spec.txt`として保存されます。

4. **リファクタリング提案の生成**:
   - 統合されたコードに対して、AIがリファクタリング提案を行います。
   - 提案は`check_refactoring.txt`として保存されます。

## 3. 非機能要件
- **パフォーマンス**: 各処理は合理的な時間内に完了すること。
- **信頼性**: 例外処理を含む堅牢なエラーハンドリングを実装すること。
- **ログ記録**: すべての活動をログに記録し、デバッグモードで詳細なログを取得可能にすること。

## 4. 技術要件
- **言語と環境**: Python 3.x
- **外部ライブラリ**: OpenAI APIライブラリ
- **設定ファイル**: `settings.ini`において、APIキーやディレクトリ情報を設定可能。
- **実行環境**: WindowsとUnix系OS上での動作を想定。
- **依存関係**: `requirements.txt`に必要なパッケージを記載。
